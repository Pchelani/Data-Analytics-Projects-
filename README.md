# Colorectal Cancer Prediction
This project predicts colorectal cancer survival using machine learning models.
Features:
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Logistic Regression model  
- Model evaluation
Technology used:
- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  
- Scikit-learn
  Dataset used:
  Dataset sourced from hospital records (CSV format).
how to run the project-
1. Clone the repository  
2. Install required libraries  
3. Run the notebook
Output-
The model achieved an accuracy of 82%.
